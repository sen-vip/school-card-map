# 학교카드 사용지도 v1.1.6

서울특별시교육청 업무추진비 공개자료를 자동으로 불러와 학교카드 사용처를 지도와 표로 정리하는 도구입니다.

## v1.1.6 변경사항

- 자동수집 성공 후 바로 보이는 **자료 확인 완료 / 지도 만들기 CTA 박스** 추가
- `지도 만들기` 버튼이 수동 입력 영역 안에 묻히지 않도록 요약 카드 위로 이동
- 자동수집 후 흐름을 `서울교육청에서 가져오기 → 자료 확인 → 지도 만들기`로 정리
- 지도 생성 후 CTA 문구를 `지도 다시 만들기`로 변경
- 수동 붙여넣기 영역을 결과/지도 아래의 보조 기능으로 이동
- 기존 기능 유지: POST 자동수집, 상세 다중 페이지 수집, 지도 제외 규칙, 카카오맵 표시

## 실행 방법

자동수집은 로컬 서버가 필요합니다.

```powershell
py server.py
```

브라우저에서 아래 주소로 접속합니다.

```text
http://localhost:5500
```

## 사용 흐름

1. 학교명과 기준월을 입력합니다.
2. `서울교육청에서 가져오기`를 누릅니다.
3. 자료 확인 완료 박스가 보이면 `지도 만들기`를 누릅니다.
4. 표시 완료 / 위치 확인 / 지도 제외 결과를 확인합니다.

## 주의

- 자동수집은 서울특별시교육청 열린서울교육 업무추진비(학교) 공개자료 전용입니다.
- 카카오 JavaScript 키는 `지도 API 설정` 접힘 영역에 저장합니다.
- 배포 후에는 카카오 개발자 콘솔의 JavaScript SDK 도메인에서 실제 배포 도메인만 남기는 것을 권장합니다.
